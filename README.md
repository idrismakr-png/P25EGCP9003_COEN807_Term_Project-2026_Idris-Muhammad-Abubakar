# COEN807 Term Project: Credit Card Fraud Detection

## Project Overview
This project applies machine learning techniques to detect fraudulent credit card transactions using a publicly available dataset. Three supervised learning classifiers (Logistic Regression, Decision Tree, and Support Vector Machine) are implemented, tuned, and compared.

## Dataset
- **Source:** Kaggle / UCI Machine Learning Repository
- **Name:** Credit Card Fraud Detection (Dal Pozzolo et al., 2015)
- **Size:** 284,807 transactions, 31 features
- **Class distribution:** 0.17% fraud, 99.83% legitimate
- **Access:** [Kaggle Link](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)

> **Note:** The dataset is not included in this repository due to size. Download it from the link above and place `creditcard.csv` in the project root folder.
